# PESI / sPESI Calculator

## Objetivo

Estratificar risco em pacientes com tromboembolismo pulmonar usando PESI completo e sPESI.

## Modelos

- PESI completo
- sPESI simplificado

O app mostra o score, a classe de risco e a leitura clinica apropriada para cada modelo.

## Referencias

- Aujesky et al. Pulmonary Embolism Severity Index
- Jimenez et al. Simplified PESI

## Rodar localmente

```bash
npm install
npm run dev
```

## Testes



## Deploy


## Aviso legal

Ferramenta de apoio a decisao. Nao substitui julgamento clinico. Estratificacao deve ser integrada a imagem, biomarcadores, estabilidade hemodinamica e contexto assistencial.
